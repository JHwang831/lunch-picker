# 🍱 점심 뭐 먹지? - 완성 버전

모든 요구사항 완벽 구현! Firebase 실시간 동기화 + 13개 밥집 + 투표 시간 제한

## ✅ 구현된 기능

1. ✅ 밥집 시스템 (메뉴→밥집)
2. ✅ 황준혁만 밥집 추가/삭제 (관리자 권한)
3. ✅ 네이버맵/카카오맵 링크
4. ✅ Firebase 실시간 동기화
5. ✅ 투표 시간 제한 (9시-12시)
6. ✅ 13개 밥집 기본 등록
7. ✅ 비밀번호 변경 실시간 반영
8. ✅ 투표 1위 실시간 강조 (🏆)
9. ✅ 카운트다운 타이머

## 🚀 빠른 시작

### 1. Firebase 설정 (5분)
1. https://console.firebase.google.com 접속
2. 프로젝트 생성 → Firestore 생성 (테스트 모드)
3. 웹 앱 등록 → SDK 설정 복사
4. `src/firebase.js`에 설정 붙여넣기

### 2. 실행
```bash
npm install
npm run dev
```

### 3. Vercel 배포
```bash
git init && git add . && git commit -m "Initial commit"
git push
```
Vercel에서 Import → Deploy!
